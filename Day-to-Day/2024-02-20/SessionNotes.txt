# JavaScript Strings:
Refer: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String

- String is sequence of characters.
- String is one of commonly used datatype.
- JavaScript allows 'Single Quotes' and "Double Quotes".
- String provides us lot of methods.
eg.,
string.length;
string.toUpperCase();
string.toLowerCase();
string.includes("__sub-string__");
string.toLowerCase().includes("__while-searching-case-sensitive-text__");
let words = string.split(" ");
