Single Page Application(SPA)
- The application is divided into separate components.
- Each component can be managed separately.
- This make it easy to maintain the component.


- Angular is a framework used to build single page applications.
- It is free and open source.
- It is TypeScript based.
- It is developed & maintained by Google.


# Prerequisite for Angular:
    - Technologies:
        - HTML
        - CSS
        - JavaScript
        - TypeScript

    - Software:
        - node
        - npm (Node Package Manager)
        - tsc (TypeScript Compiler)
        - Angular CLI (Command-Line Interface)
        - VS Code (IDE)


# Follow following commands to create first Angular App:
1. npm install -g @angular/cli
2. Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
3. ng new my-app
4. cd my-app
5. ng serve

# Command to open App:
> ng serve --open
> The --open (or just -o) option automatically opens your browser to http://localhost:4200/


# Server Path: localhost:4200
